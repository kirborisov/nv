For update npm
sudo npm install npm -g

For install node with npm
1. Download last version from https://nodejs.org
2. sudo apt install xz-utils
3. sudo tar -xvf name_of_file
4. sudo cp dir_name/{bin,include,lib,share} /usr/

       https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim'
